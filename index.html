About the Project

AgriTwin-ZM is a simulation system built in Java to model agricultural supply, demand, and revenue for crops like maize and tomatoes. It provides both a console-based interface and a graphical Swing GUI.

Key Features

  • Input crop data (production, storage, spoilage, price, demand)
  • Real-time supply and demand calculations
  • Revenue estimation for maize and tomatoes
  • Graphical charts and reports
  • Export simulation results to text files
